## 🔍 Find Information By Topic

### I want to...

#### 🚀 Get Started
1. Read **QUICKSTART.md**
2. Run `pnpm install && pnpm dev`
3. Open http://localhost:3000

#### 🎨 Change Colors
1. Edit `app/globals.css`
2. Update `--primary` variable
3. Save and auto-reload

#### 🚙 Add or Edit Vehicles
1. Edit `app/page.tsx` (lines ~24-80)
2. Update vehicle data
3. Add new vehicle to array

#### 🖼️ Update Images
1. Add image to `public/` folder
2. Update image path in `app/page.tsx`
3. Component handles responsive sizing

#### 💬 Change Chatbot
1. Edit `components/ChatbotWidget.tsx`
2. Update chatbot URL (lines ~34, 56)
3. Customize header text

#### 📱 Test Mobile
1. Open F12 (DevTools)
2. Press Ctrl+Shift+M (Toggle device)
3. Select different devices
4. Test interactions

#### 🚀 Deploy
1. Follow **SETUP.md** (Deployment section)
2. Push to GitHub
3. Connect to Vercel
4. Auto-deploys on push

#### 🐛 Debug Issues
1. Check **QUICKSTART.md** (Troubleshooting)
2. Open F12 Console
3. Look for errors
4. Fix and reload

#### ✅ Pre-Launch
1. Use **LAUNCH_CHECKLIST.md**
2. Check all items
3. Fix any issues
4. Deploy!

## 🚀 Deployment Options

### Recommended: Vercel
1. Push code to GitHub
2. Connect to Vercel
3. Auto-deploys on push
4. Free SSL & CDN included

### Alternative: Self-hosted

3. Port: 3000 (configurable)
4. Any server compatible

1. Build image

## 🎯 Customization Examples

### Change Vehicle Prices
**File**: `app/page.tsx` (line ~29)
```typescript
price: '1.099.000.000',  // Change this number
```

### Change Colors
**File**: `app/globals.css` (line ~1)
```css
--primary: oklch(0.55 0.2 30);  /* Change to your color */
```

### Change Chatbot URL
**File**: `components/ChatbotWidget.tsx` (line ~34)
```typescript
src="https://YOUR_NEW_CHATBOT_URL"
```
